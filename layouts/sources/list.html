Data Sources

FCN_ERA5_DATA_V0

FourCastNet ERA5 data from the project Globus instance, originally sourced from the Copernicus Climate Change Service Climate Data Store.

ERA5 data for FourCastNet ↗

MERRA-2 Datasets

Public NASA atmospheric reanalysis datasets used for weather and Earth system research.

NASA MERRA-2 archive ↗

WPS_GEOG

Static geographical data from OpenWFM used for WRFxPy and WRF preprocessing workflows.

Download WPS_GEOG.tbz ↗

WFIGS / Fire Perimeters

CAL FIRE eGIS perimeter data used for wildfire perimeter reference and comparison work.

CAL FIRE wildfire perimeters ↗

The CAL FIRE / WFIGS perimeter data is dynamic and should be treated as a reference source, not a legal record. Accuracy, completeness, and downstream use remain the responsibility of the data user.
